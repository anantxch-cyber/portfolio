01 — computer vision
AI Phone Detection & Alert System 📵

Real-time webcam system using YOLOv8 that detects if you picked up your phone and plays a loud alarm. Built this because willpower wasn't cutting it during study sessions.

"The alert literally says 'PHONE RAKHHHH NEECHE!' in the code — English just doesn't have the urgency this needed." 😂

How it works → YOLOv8n detects phone in webcam → PyGame plays audio alert. Frame-skipping every 3rd frame keeps it smooth on a basic laptop without GPU.
Proud of → It actually works. Sub-100ms detection. I use it myself while studying.

02 — gesture recognition
Monkey Meme Reaction System 🐵

Wave at the camera and get a matching monkey meme in the corner of your screen. Thumbs up → happy monkey. Just sitting there → the famous neutral stare. Yes, this is real CV.

"My friends use this daily. One guy has it on during online classes. I take no responsibility." 🙏

How it works → MediaPipe tracks 21 hand landmarks in real-time → gesture classified → matching meme image overlaid via OpenCV.
What I learned → Hand landmark coords, confidence thresholding, frame compositing, MediaPipe multi-model pipeline.
